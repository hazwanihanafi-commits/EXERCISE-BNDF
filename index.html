Overview

This portal provides evidence-informed 8-week exercise prescriptions tailored to BDNF Val66Met genotypes. Each plan combines aerobic, resistance, cognitive-motor, and mind-body elements.

Compare Genotypes

Val/Val

  • 4–5 sessions/week • 20–30 min
  • Moderate–vigorous intensity (HIIT friendly)
  • Emphasis: variation, skill learning, performance

Met allele

  • 5–6 sessions/week • 30–45 min
  • Light–moderate steady-state intensity
  • Emphasis: consistency, cognitive engagement, recovery

  • Friday: Endurance Intervals (Steady)

  • Sunday: Rest / Breathing

Adventure Mode (10–15 min)

Forest Run

March/jog 60s — count every 3rd step.

Mountain Climb

8×Squats → 8×Knee lifts → 8×Lunges. Reverse order on round 2.

Puzzle Bridge

One-leg balance 10s each side + name 5 items.

Enemy Encounter

Respond to random cues mapped to movements.

River Crossing

Plank 20–30s + seated leg lifts; visualize and recall a path.

Victory Stretch

Deep breathing and reflection.
